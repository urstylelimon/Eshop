🛍️ Loriva E-commerce Platform

A full-featured Django-based e-commerce system with inventory, orders, and profit/loss tracking — designed for business growth and management.

🚀 Features

👥 User information and cart system used cache memory

🛒 Shopping Cart & Orders – Add to cart, confirm, and track orders

📦 Inventory Management – Real-time stock updates and low-stock alerts

💰 Profit & Loss Dashboard – Auto profit calculation and monthly reports

💻 Responsive UI – Clean, modern, mobile-first interface


Installation Guide

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt

python manage.py runserver
